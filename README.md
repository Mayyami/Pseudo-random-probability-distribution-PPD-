# Pseudo-random-probability-distribution-PPD-
This code show how random abilities(like critical hit, stun and more modification of attack) in games like League of Legends, Dota2, Warcraft3, Vermintide2 worked
Чтобы начать использовать просто запустите файл PPD.py(через pyhton(Default) или python 3.10) и следуйте инструкциям выводящимся в консоль
